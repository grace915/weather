# weather

## 앱 설명
- 날씨를 알려주는 앱
- 날씨에 따라 배경화면이 바뀜
- 풍속, 기온, 습도를 알려줌
- volley, Glide 라이브러리를 사용
- openweathermap api를 사용하여 해당 지역의 날씨 정보를 받아옴
